# Strumenti e Script Autorizzati

Per garantire la sicurezza, l'affidabilità e la coerenza di ogni screenshare, è fondamentale adottare una **lista ufficiale e tassativa** di strumenti e script che possono essere utilizzati durante un controllo.

L'uso di qualsiasi software non presente in questo elenco è **severamente proibito**, a meno che non venga esplicitamente autorizzato dalla leadership del team in circostanze del tutto eccezionali.

Questa politica serve a raggiungere tre obiettivi fondamentali:

* **Proteggere il Player:** Utilizziamo solo software testato e verificato, noto per essere sicuro, privo di malware e non invasivo. Questo tutela l'integrità del sistema del giocatore.
* **Standardizzare le Procedure:** Assicura che ogni staffer utilizzi gli stessi strumenti, rendendo i controlli più equi, i risultati più consistenti e il processo di revisione più semplice.
* **Garantire la Validità delle Prove:** Le prove raccolte con strumenti non autorizzati o sconosciuti potrebbero essere considerate inaffidabili e, di conseguenza, non valide in caso di appello. L'uso di un set di strumenti approvato rafforza l'integrità delle nostre conclusioni.
